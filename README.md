# Passoword-Generator-WIP
This is an password generator that either creates as long of a standard password you want(Not ideal if you want 100 length) or generates a seed phrase password ranging from 12 to 24 words.

This is an Work In Progress (WIP) meaning that currently in 22/02/2026 it's not that good and is not advised to use Normal password gen if you want the strongest non-seed phrase password
